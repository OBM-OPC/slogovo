# Product navigation

Slogovo has six named learning areas with one clear daily action:

- **Lernen** owns the course and the primary **Heute lernen** adaptive session.
- **Wiederholen** contains only vocabulary currently due for spaced repetition.
- **Sprechen** contains transparent listen-and-repeat self-review without automated pronunciation claims.
- **Fehler** contains productive recall for vocabulary with recorded mistakes.
- **Wortschatz** owns browsing, search, module categories, cards, typing, and word building.
- **Fortschritt** owns learning-state and mastery reporting.

The mobile bottom bar stays at four high-frequency destinations: Lernen, Wiederholen, Wortschatz, and Fortschritt. Sprechen and Fehler remain one tap away in the Lernen hub, where all six areas are visible. Profile and settings are utility actions in the hub header. Alphabet and grammar are secondary reference tools, not competing primary learning entry points.

This removes the previous ambiguous bottom bar of Bulgarian labels and utilities, keeps **Heute lernen** as the only visually dominant dashboard action, and avoids separate routes that launch nearly identical vocabulary trainers.
